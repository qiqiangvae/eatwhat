# 今天吃什么 - 设计文档

## 概述

一款解决"选择困难症"的趣味决策工具，帮助用户在多个选项中快速做出决定。

## 技术栈

- React 18 + Vite
- TailwindCSS
- localStorage 持久化
- 移动端优先设计


## 三大决策模式

### 🎲 骰子模式

- 显示多颗彩色骰子（可调数量：1-6颗）
- 点击"摇一摇"按钮触发动画
- 骰子翻滚 1-2 秒后停止
- 根据点数之和确定选中的菜单项

### 🎡 转盘模式

- 圆形转盘，均分 N 份（N = 当前筛选后的选项数量）
- 每份显示一个选项名称
- 点击"转！"按钮
- 转盘旋转 3-5 秒，物理减速效果
- 指针指向的选项为结果，带高亮动画

### ⚔️ 淘汰模式

- 选择参与数量（从筛选结果中随机选取：8/16/32）
- 展示对阵图（两两PK）
- 用户点击选择每轮的晋级者
- 动画展示对决结果
- 最终冠军为选择结果

## 页面详情

### Tab 1: 决策页

- 顶部模式切换器：骰子 / 转盘 / 淘汰
- 模式切换时平滑过渡动画
- 下方显示当前筛选的选项数量

### Tab 2: 菜单页

- 顶部横向滚动标签栏
- 点击标签多选筛选
- 下方网格展示匹配的菜单项
- 点击任意选项可单独"决定这个"

### Tab 3: 记录页

- 列表展示历史决策记录
- 每条记录显示：时间、模式、结果
- 支持清空历史

```

辅助色1:  #4ECDC4 (薄荷绿)

浅色文字: #636E72
```

### 设计元素

- 圆角：16-24px（卡片、按钮）
- 阴影：柔和投影 `0 4px 20px rgba(0,0,0,0.1)`
- 动效：弹跳、缩放、旋转，强调趣味性
- 字体：系统圆体字体
